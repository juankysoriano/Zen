# Asset Attribution

The original Zen flower sprites, UI icons, brush texture, and `zen.mp3` came
from the legacy Android project in this repository.

Additional imported assets:

- `assets/audio/reaching_the_sky.ogg` and `assets/audio/reaching_the_sky.m4a`:
  "Reaching The Sky" by Alexander
  Nakarada, downloaded from Wikimedia Commons via `Special:Redirect/file`.
  License: Pixabay Content License as listed on the source page. The `.m4a`
  version was locally converted from the downloaded `.ogg` for iOS playback.
- `assets/audio/placid_ambient.ogg` and `assets/audio/placid_ambient.m4a`:
  "Placid Ambient" by MusicLFiles, downloaded from Wikimedia Commons via
  `Special:Redirect/file`. The `.m4a` version was locally converted from the
  downloaded `.ogg` for iOS playback.
- `assets/images/blue_flower_commons.png`: "Blue Flower Transparent
  Background.png" by The People's Internet / JOGOS Public Assets, Wikimedia
  Commons. License: CC0 1.0 Universal Public Domain Dedication.
- `assets/images/addisonia_commons.png`: "Addisonia (PLATE 075)
  (8577495976) inflorescence transparent background.png", Missouri Botanical
  Garden, Peter H. Raven Library, via Wikimedia Commons. License information on
  Commons lists CC BY 2.0 and public-domain source scan status. Resized and
  converted to transparent background for in-app sprite use.
